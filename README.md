#  — Portfolio (Flask)

Mobile-first, professional, English-only, no header. Photo supported (use static/images/photo.jpg).

Quick start:
  pip install -r requirements.txt
  python app.py
  # open http://127.0.0.1:5000

To add your photo:
  - Put your image at: static/images/photo.jpg (JPEG recommended, ~300–600px square)
  - The site will automatically use it; otherwise it falls back to avatar.svg
